1. Install Zadig (for the printer driver — one time only):
- Download from https://zadig.akeo.ie
- Plug in the printer
- Run Zadig as Administrator
- Select "POS58 Printer" → click "Install Driver"
- Done — never needs to be repeated unless Windows is reinstalled

2. Create a desktop shortcut that always runs as Admin:
- Right-click POS_System.exe → Send to → Desktop (create shortcut)
- Right-click the shortcut → Properties → Advanced → check "Run as Administrator" → OK
